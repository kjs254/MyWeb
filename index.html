    3️⃣ HTML

    HTML은 Hyper Text Markup Language의 약자로 간단히 말하자면 웹사이트를 기술하는 언어입니다.

    HTML에 관한 구체적인 설명은 다른 매체를 통해 자세히 알아보시는 것을 추천합니다. 저는 생활코딩의 HTML 강좌 영상을 개인적으로 선호합니다.

    F12 혹은 Ctrl + U를 입력하면 이 페이지를 작성한 HTML을 확인할 수 있습니다. 이를 바탕으로 이 문서에서 쓰인 요소들을 소개하겠습니다.

    💎 상위 태그 및 헤드 태그들

    • <!DOCTYPE html> : 해당 문서의 타입을 정의 (HTML 5)
    • <html> : HTML 문서 (HTML)
    • <head> : 문서에 대한 정보를 기술 (타이틀, 언어, 링크 등)
    • <body> : 문서에 작성될 내용을 기술 (태그, 글, 이미지 등)
    • <title> : 해당 웹페이지의 제목 (metadata)
    • <meta> : 해당 문서에 대한 정보 (metadata)
    • <link> : 외부 소스와의 관계 (CSS 등)
    • <script> : 사이드 스크립트 (JavaScript 등)

    💎 바디 태그들

    • <H1>, <H2>, ... , <H6> : 제목 (heading)
    • <p> : 본문 (paragraph)
    • <a> : 하이퍼 링크 (anchor)
    • <ul> : 순차 리스트 (undered list)
    • <ol> : 비순차 리스트 (ordered list)
    • <li> : 리스트 요소 (list item)
    • <table> : 표 (table)
    • <tr> : 표의 한 행 (table row)
    • <th> : 표의 제목이 되는 셀 (table header)
    • <td> : 표의 데이터 셀 (table data cell)
    • <img> : 이미지 (image)
    • <div> : 구역 (division)
    • <hr> : 수평선 (horizontal rule)
    • <br> : 줄바꿈 (break line)
    • <figure> : 독립된 콘텐츠 (figure)
    • <figcaption> : 콘텐츠의 캡션 (figure caption)
    • <footer> : 하단영역 (footer)
    • <iframe> : 현재 HTML 문서에 다른 문서를 포함 (inline frame)
    • <footer> : 하단영역 (footer)

    외에도 다양한 태그들이 존재합니다. TCP SCHOOL에서 HTML5의 태그들을 만나보실 수 있습니다.

    4️⃣ CSS

    CSS란 Cascading Style Sheets의 약자로 사용자에게 문서를 표시하는 방법을 지정하는 언어입니다. 개인적으론 디자인 요소를 담당한다고 생각합니다. 이 CSS를 이용하여 글자의 폰트나 크기, 색상을 바꾸거나 화면에 어떻게 나타낼 지 결정할 수 있습니다.

    이 페이지에서는 style.css파일을 작성하고 <link> 태그를 통해 좌측의 사이드바, 본문의 레이아웃, 이미지의 크기 등 거의 모든 요소에 CSS를 적용하였습니다. F12를 누른 뒤 CSS 폴더의 style.css 파일을 확인하여 이 페이지의 모든 CSS를 확인할 수 있습니다.

    💎 자주 사용된 CSS

    • width : 요소의 너비
    • height : 요소의 높이
    • display : 요소가 보여지는 속성 (inline, block, none)
    • margin : 바깥쪽 여백, 상하좌우 지정가능 (top, bottom, left, right)
    • padding :안쪽 여백, 상하좌우 지정가능 (top, bottom, left, right)
    • border : 요소의 테두리
    • color : 요소의 색깔
    • background-color : 요소의 배경색깔

    CSS에 대한 디테일한 정보 역시 개인적으로 생활코딩의 CSS 수업을 추천합니다.

    CSS에서 중요한 선택자에 대한 이해가 굉장히 중요합니다. tag, class, id 등에 대하여 원하는 요소를 선택하는 방법은 다양하지만 글로 이해하는 것보다 게임을 통해 이해하는 것이 몸에 익히기 좋기 때문에 총 32개의 레벨로 구성된 CSS Diner 게임을 소개해드립니다.


    가리키는 요소만을 선택하는 선택자를 선언하면 스테이지가 넘어가는 게임입니다. 다양한 선택자 기법을 연습하기 좋은 게임이라고 생각합니다.

    5️⃣ JavaScript

    HTML 문서는 기본적으로 정적인 문서 파일이므로 파일의 내용이 변경되거나 이벤트가 발생하지 못한다. 이를 가능하게 해주는 스크립트 언어 혹은 프로그래밍 언어업니다. 버튼을 누르면 계산을 해주거나, 알람을 띄우거나, 문서내용을 변경하는 것이 수업에서 배운 내용입니다.

    CSS와 마찬가지로 F12를 누른 뒤 js 폴더의 script.js 파일을 확인하여 이 페이지의 자바스크립트를 확인할 수 있습니다.

    💎 사용된 자바스크립트 함수

    1. 스크롤이 내려가면 ▲ 버튼이 우측 하단에 나타나는 함수

      2. - 조건문을 통하여 스크롤이 움직일 때, scrollTop이 300보다 크다면 display를 block으로 설정하도록 설계

    2. ▲ 버튼을 클릭하면 스크롤이 맨 위로 올라가는 함수

      3. - addEventListener()를 통해 버튼을 클릭하면 window.scrollTo()를 실행하는 함수가 발생, 목적지의 top을 0으로 지정

    3. 사이드바의 목차를 클릭하면 지정된 파트로 스크롤이 이동하는 함수

      4. - section ID를 가진 <a>태그를 탐색, 이를 클릭하면 해당 ID를 가진 요소를 클릭하면 요소가 있는 곳으로 스크롤이 이동하는 scrollIntoView()함수가 발생하도록 addEventListener()를 설정
